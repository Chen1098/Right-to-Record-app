# Right-to-Record-app
The IOS app Right to Record, which is an app that records and uploads at the same time, this can protect your video footages so that even if your phone was destoried when recording, you'll still have most of the footages.


Inside Secure Camera is all the swift code for the ios app


Go to windows, create a server file and then just python server.py to run the server

then install the ios app on your phone using xcode.

then it works!
